# Concept Maps

A concept map is a diagram that shows relationships between ideas using labeled nodes and arrows.

Unlike outlines — which are hierarchical — concept maps allow arbitrary connections,
making them better suited for capturing non-linear knowledge.

## Relation to linking

[[2026053102]] describes how linking ideas creates emergent structure.
Concept maps make that structure visible at a glance.

## Building one

Start with a central concept. Ask "what is related to this?" for each node.
Label the arrows: "causes", "is a type of", "supports", "contradicts".
