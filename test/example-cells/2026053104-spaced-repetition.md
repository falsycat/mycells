# Spaced Repetition

Spaced repetition is a learning technique that schedules reviews at increasing intervals.
The core idea: review material just before you are about to forget it.

Key insight: forgetting is not the enemy — forgetting and then recalling strengthens memory traces.

## The Forgetting Curve

Hermann Ebbinghaus showed that without review, retention drops sharply within days.
Spaced repetition flattens this curve by inserting reviews at the right moments.

## Connection to note-taking

When combined with [[2026053101]], spaced repetition turns a note archive into an active learning system.
Writing a note is only the first step — revisiting it on a schedule is what drives retention.
