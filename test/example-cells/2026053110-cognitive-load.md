# Cognitive Load

Cognitive load is the mental effort required to process information.

Working memory is limited — typically 4 ± 1 chunks at a time.
When load exceeds capacity, comprehension breaks down.

## Three types

- **Intrinsic**: difficulty inherent to the material.
- **Extraneous**: difficulty caused by poor presentation.
- **Germane**: effort that builds lasting schemas.
