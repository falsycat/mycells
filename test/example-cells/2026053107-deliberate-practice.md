# Deliberate Practice

Deliberate practice is focused, effortful work at the edge of current ability,
guided by immediate feedback.

Coined by K. Anders Ericsson, it distinguishes expert-level improvement from
mere repetition.

## Core elements

1. A specific, stretching goal.
2. Full concentration — no autopilot.
3. Feedback loops that reveal errors quickly.
4. Targeted repetition of the weak spots.

## Connection to spaced repetition

[[2026053104]] handles *when* to revisit material.
Deliberate practice addresses *how* — the quality and focus of each review session.
