# Linking Ideas Instead of Organizing Them

Most note-taking systems ask you to file notes into folders or tag them by category. The Zettelkasten approach asks something different: link each new note to an existing one that it relates to.

The difference matters. A folder says where a note lives. A link says how an idea connects. Folders create silos; links create paths.

When you write a new note, the question isn't "what category does this belong to?" but "what other thought does this continue, contradict, or build on?" That question forces you to actually think about the idea rather than just store it.

Over time the links themselves become meaningful. A note with many incoming links is a hub — a concept that keeps showing up in your thinking. Noticing that is valuable signal.
